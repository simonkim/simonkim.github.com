Setup progress so far

Thursday, November 22 2012 12:34 PM

Steps I've followed so far

Downloading source code repository (git) ...bash $ git clone git@github.com:rhiokim/haroopress.git samplepress ...

Initialization $ make init

Note: My git global configuration core.autocrlf was set to true. I guess this was the reason make init failed to complete due to failure of executing ./bin/setup.js that has the first line #!/usr/bin/env node followed by a hidden LF(M in vi). Anyhow, changing core.autocrlf to input and starting over from git cloning fixed this issue
